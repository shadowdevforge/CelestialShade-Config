Celestial Shade uses the `SUPER` key (Windows Key) as the main modifier. The configuration supports **Vim Motions** (HJKL) alongside standard Arrow keys for maximum accessibility.

## ü™ü Window Management

| Action | Keybind | Description |
| :--- | :--- | :--- |
| **Close Window** | `Super + Q` | Kill active window |
| **Float/Tile** | `Super + V` | Toggle floating state |
| **Fullscreen** | `Super + F` | Toggle fullscreen |
| **Focus** | `Super + H/J/K/L` | Vim-style focus navigation |
| **Move** | `Super + Shift + H/J/K/L` | Move window in direction |
| **Dwindle Split**| `Super + \` | Toggle split direction |
| **Pseudo Tile** | `Super + P` | Dwindle pseudo-tiling mode |
